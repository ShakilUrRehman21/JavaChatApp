#  JavaChatApp

A real-time chat application built using **Spring Boot** and **Firebase**, featuring user authentication and live message synchronization.

This project demonstrates backend integration with Firebase along with real-time communication and authentication handling.

---

##  Features

*  User authentication
*  Real-time messaging
*  Multiple users support
*  Firebase message storage
*  Instant data synchronization
*  Spring Boot backend integration

---

##  Tech Stack

* **Java**
* **Spring Boot**
* **Firebase (Realtime Database / Firestore)**
* **HTML, CSS, JavaScript**
* **Maven**

---

##  How It Works

1. User registers/logs in.
2. Authentication is validated.
3. Messages are sent to Firebase.
4. Firebase synchronizes messages in real time.
5. All connected users receive updates instantly.

---

##  Run Locally

```bash
git clone https://github.com/ShakilUrRehman21/JavaChatApp.git
cd JavaChatApp
mvn spring-boot:run
```

Open in browser:

```
http://localhost:8080
```
